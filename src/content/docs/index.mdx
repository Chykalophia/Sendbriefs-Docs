## How these docs are organized

- **Getting started** is a sequential track. Read it in order the first time.
- **Core features** is a reference. Skim to the page you need.
- **Processes & flows** is recipes for common workflows. Pull one up when you're setting up that workflow for the first time.
- **Billing & plans** is reference for the subscription side.
- **API & developers** is a placeholder for now — public API is in development.

## Reading comfort

There's a Comfort Panel button in the bottom-right of every page. Use it to:

- Adjust text size (Small / Medium / Large)
- Adjust density (Compact / Comfort / Spacious)
- Enable a dyslexia-friendly font (Atkinson Hyperlegible)
- Enable a high-contrast theme
- Reduce motion
- Toggle Skim mode (collapse long prose paragraphs to one line each — hover to expand)

Your choices save on this device. They follow you across pages.
